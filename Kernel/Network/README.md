
# OBOS

A Custom 32 bit operating system.
## Authors

- [@omersaban](https://github.com/Frogithezhrda)
- [@barakshalom](https://github.com/barakshalom)


## Deployment

To deploy this project run

```bash
  make clean
  make
  ./j or ./r or ./run.sh
```


## Features

* Bootloader
* Interrupt handling
* Paging (virtual memory)
* Heap allocator
* Round-robin scheduler
* Basic userspace
* System calls (with basic wrapper)
* VGA graphics driver
* Keyboard driver
* Timer driver
* RTC driver
* Console driver
* Basic sound driver
* Custom file system
* Basic network stack (IPv4, ICMP)
* Login page
* Snake game
* Calculator
* System stats tool
